# EstateGPT

An AI-powered Real Estate CRM and Sales Copilot built with FastAPI, React, PostgreSQL, n8n, and Large Language Models.

## Vision

To automate lead management, customer communication, property recommendations, document intelligence, and sales workflows for real estate companies.

## Tech Stack

- FastAPI
- React
- PostgreSQL
- SQLAlchemy
- n8n
- OpenAI/Gemini
- ChromaDB
- Docker
